## Features

### 为什么要用代码写PPT呢？

- 源代码最终被编译成html,css,js。Web应用有很好的视觉表现力

- 用*coding*的方式写代码可以让我们忽略PPT制作过程中一些无关紧要的细节，直接使用或者定义样式模版

- 代码更容易维护和分享
